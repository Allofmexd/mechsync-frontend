# Documentación frontend MechSync

Esta carpeta contiene referencias visuales y guías para implementar las vistas del frontend de MechSync.

El contenido de `docs/` no es código productivo. Los archivos SVG exportados desde Figma sirven como referencia visual para analizar layouts, componentes, estados y decisiones de experiencia de usuario; no representan componentes finales de la aplicación.

Las implementaciones reales deben desarrollarse dentro de `src/`, respetando la arquitectura y el stack actual del frontend.

## Estructura

- `figma/views/`: vistas SVG exportadas desde Figma.
- `figma/notes/`: notas manuales sobre diseño, componentes y UX.
- `implementation/`: guías y correspondencias entre diseños, rutas, features y endpoints.
